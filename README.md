## 진행 방법
약 1달 간, 대면으로 진행하고, 이후 진행은 issues를 통해 진행합니다.

## 목표: 자바와 친해지기
새로운 언어를 시작하긴 쉽지 않습니다.
그렇기 때문에 기본 문법을 먼저 시작하는 것이 언어를 이해하는 데 큰 영향을 미칩니다.
**하지만** 이번 스터디에선 새로운 언어에 대한 두려움(문법을 모두 이해해야 한다는 무서움)을 없애고, 재미를 가지는 것에 조금 더 집중 할 것입니다.
